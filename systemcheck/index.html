Systemcheck

Hier können Sie überprüfen, ob Ihr Computer technisch in der Lage ist, golfdigital.tv zu empfangen.

Sie benötigen einen Multimediarechner (PC oder Mac), der idealerweise nicht älter als 3 Jahre ist.

Sie benötigen eine DSL-Verbindung zum Internet.

Jetzt Systemcheck starten

Technische Mindestanforderungen

Komponenten   Anforderung
Personal Computer unter Windows   X86- oder x64-Prozessor mit 1 GHz oder höher sowie mind. 512 MB RAM
Mac OS 10.4.8+ (PowerPC)   PowerPC G4-Prozessor mit 1 GHz oder höher sowie mind. 512 MB RAM
Mac OS 10.4.8+ (Intel-basiert)   Intel Core Duo-Prozessor mit 1,83 GHz oder höher sowie mind. 512 MB RAM
